> 문자열 잘라내기



'hello world' 에서 일부만 자르고 싶을 때, 

```python
>>> a = 'hello world'
>>> a[0:5]
'hello'
>>> a[6:11]
'world'
```

`a[0:5]` 와 `a[:5]` 는 같은 문자열('hello')을 출력한다.

또한

`a[6:11]` 과 `a[6:]` 또한 같은 문자열('world')을 출력한다.



> 음수 인덱싱

```python
>>> a[6:-2]
'wor'
```

`a[6:-2]`에서 `-2`는 뒤에서 2번째까지 잘라내겠다는 것을 의미한다 따라서 맨 뒤에 문자 ld가 잘린것을 볼 수 있다.
