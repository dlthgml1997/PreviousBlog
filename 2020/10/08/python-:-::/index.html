[파이썬] 사칙연산 '/'와 '//'의 차이

2020-10-08

알고리즘을 푸는 와중,

음수를 양수로 나눌 때는 C++14의 기준을 따른다. 즉, 양수로 바꾼 뒤 몫을 취하고, 그 몫을 음수로 바꾼 것과 같다.

이러한 조건이 있는 문제를 만났다.


예를 들어, -4 를 3으로 나눈다고 할 때

-4 * -1 // 3 * -1 이렇게도 조건을 만족할 수 있지만, int(-4/3) 이렇게 하는 것 또한 조건을 만족할 수 있다.


그 이유가 궁금해서 / 와 // 의 차이를 알아보았다.


/ 와 // 의 차이

  • / : 나누기

  • // : 나누기 연산 후 소수점 이하의 수를 버리고, 정수 부분의 수만 구함.


내가 궁금했던 건 / 와 // 가 음수에 쓰일 때 였다.

음수를 양수로 나눌 때 !

// 는 내림을 하고,

/ 는 올림을 한다.


결론 : C++14 기준을 따라 음수를 양수로 나눌 때, int(음수/양수) 를 사용할 수 있다.
