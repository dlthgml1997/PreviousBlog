Mac Operation not permitted 오류 해결 - [Errno 1] Operation not permitted Program ended with exit code: 2

2020-11-12

xcode에 python 사용 설정을 마친 후에 build를 했는데 콘솔에 아래와 같은 에러가 발생했다.

실행 권한이 없어서 파일을 열 수 없다는 의미이다.


해결 방법은

설정 > 보안 및 개인 정보 보호 > 전체 디스크 접근 권한 에 들어가서 python3를 추가해주면 된다.

(+ 버튼을 누르고 shift + command + G 를 눌러 /usr/bin 폴더로 접속해 python3 를 클릭해서 추가했다. )
