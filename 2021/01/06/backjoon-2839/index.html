[백준알고리즘] 2839: 설탕 배달 (파이썬)

2021-01-06
  • 문제 링크: https://www.acmicpc.net/problem/2839

풀이

  • 설탕을 정확하게 n 킬로그램 배달할 수 있을 때 (3과 5의 조합으로 나누어 떨어질 때), 5 킬로그램 봉지가 많을 수록 배달하는 봉지의 개수가 최소가 된다.
    • 3 킬로그램 봉지를 하나씩 증가시키는 반복문 안에서 5 킬로그램 봉지를 하나씩 증가시키는 반복문을 실행한다.
      • 이때, range는 설탕 n 키로그램을 담는 최대 봉지 개수( n // 3kg 또는 5kg ) 까지로 설정한다.
    • 반복문이 실행될 때마다 3키로그램 봉지 개수(i) * 3 + 5 키로그램 봉지 개수(j) * 5 값이 n 과 같은지 비교한 후, 같으면 i + j 를 print 하고 종료한다.
  • 반복문을 모두 돌았다는 것은 3과 5로 나누어지지 않는다는 것을 의미하기 때문에 -1 을 print 한다.
