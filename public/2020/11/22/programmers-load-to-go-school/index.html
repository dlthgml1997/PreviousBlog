[프로그래머스] 등굣길 - 동적계획법(Dynamic Programming) 파이썬

2020-11-22
  • 문제 링크: https://programmers.co.kr/learn/courses/30/parts/12263 > 등굣길

풀이

  • 다이나믹 프로그래밍 은 이전에 계산해둔 값을 저장해두고, 다음번에 같은 계산이 나왔을 때 재계산하는 것이 아닌 저장해둔 값을 활용하는 것이다.

  • 평소 길찾기 문제를 풀 때처럼 바로 위와 왼쪽 칸에 있는 값을 더해가면 된다.

    • 이때, 집 위치와 물 웅덩이 위치에서는 길찾기를 위한 덧셈을 패스한다. (continue)

  • maze[-1][-1] : 마지막 인덱스로 학교 위치를 의미한다.

  • 문제 요구사항처럼 마지막에 1,000,000,007 로 나눈 나머지 값을 return 한다. (안하면 효율성 테스트 0점 맞는다.)
